# Quick PWA Installation Guide

**Note:** PWA requires HTTPS connection (use a reverse proxy). Works on localhost for development.

## Mobile Devices

### iPhone/iPad (iOS)
1. Open Family Chores in **Safari**
2. Tap the **Share** button (square with arrow)
3. Scroll and tap **"Add to Home Screen"**
4. Tap **"Add"**
5. Find the app icon on your home screen

### Android
1. Open Family Chores in **Chrome** or **Edge**
2. Tap the menu (⋮) in the top right
3. Tap **"Add to Home Screen"** or **"Install app"**
4. Tap **"Install"** when prompted
5. Find the app icon in your app drawer

## Desktop/Laptop

### Chrome/Edge/Brave
1. Open Family Chores in browser
2. Look for install icon (⊕) in the address bar
3. Click **"Install"**
4. App opens in its own window
5. Launch from:
   - **Windows**: Start Menu
   - **Mac**: Applications folder or Launchpad
   - **Linux**: Application menu

### Firefox
- Firefox supports offline functionality but does not offer install prompts
- Use as a regular website or try Chrome/Edge for full PWA experience

## Benefits of Installing

✓ **Faster access** - Launch directly from home screen/desktop  
✓ **No browser clutter** - Runs in standalone window  
✓ **Native feel** - Looks and behaves like a regular app  

## Uninstalling

### Mobile
- Long-press app icon → **"Remove"** or **"Uninstall"**

### Desktop
- **Windows**: Right-click app icon in Start Menu → **"Uninstall"**
- **Mac**: Find in Applications → drag to Trash
- **Browser**: Settings → Apps → Find Family Chores → **"Uninstall"**
